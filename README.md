# 🧠 BOB - Cerebro de CLIENDER

**Ubicación:** Disco duro 1TB

## Estructura

```
CLIENDER-BOB/
├── CLAUDE.md              ← Identidad de BOB (leer al inicio)
├── BOB-PERSONALITY.md     ← Personalidad completa
├── README.md              ← Este archivo
├── memory/                ← Memoria a largo plazo
├── proyectos/             ← Proyectos activos
├── clientes/              ← Info de clientes
├── aprendizajes/          ← Lo que BOB aprende
├── herramientas/         ← Configuraciones
├── referencias/          ← Manuales, docs
└── scripts/              ← Automatizaciones
```

## Cómo usar

1. **Para iniciar a BOB:**
   ```bash
   cd ~/CLIENDER-BOB
   claude
   ```

2. **Al inicio, BOB lee:**
   - CLAUDE.md (identidad)
   - BOB-PERSONALITY.md (personalidad)
   - memory/ (contexto)

3. **Durante el trabajo:**
   - Todo importante → guardar en carpetas correspondientes
   - Antes de terminar → resumir en memory/

## Reglas

- **BOB siempre sabe** que su cerebro está en ~/CLIENDER-BOB/
- **Todo importante** → archivo, nunca solo en memoria
- **BOB es leal** a CLIENDER
